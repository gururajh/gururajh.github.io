Created and maintained a robust E2E test automation framework using **Playwright and TypeScript**. This framework was instrumental in accelerating regression testing for a sophisticated cloud-based mobile/web healthcare application.

## Key Highlights
*   **Architecture**: Built a custom, scalable Page Object Model (POM) to handle complex, heavily dynamic UI components.
*   **API Integration**: Combined UI testing with API validations using Postman and Playwright's built-in request context to ensure full data integrity across services.
*   **CI/CD Pipeline**: Integrated the automated execution into the deployment pipeline (Jenkins/Azure DevOps), ensuring build quality before staging deployments.
*   **Impact**: Increased automated test coverage by 20%, drastically reducing the manual effort required for regression testing during Agile sprint cycles.
